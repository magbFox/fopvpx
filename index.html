ABOUT SERVER

FOPVPX 서버만의 한계 돌파형 독창적 콘텐츠 구성을 소개합니다

오직 컨트롤과 생존력으로 증명하는 야생 서바이벌 PVP

magbFoxx_FOPVPX는 시청자들과 실시간으로 소통하며 극강의 생존 본능을 가리는 하드코어 시참 서버입니다.
안전 지대에서 이루어지는 단순한 PVP를 넘어, 무자비하고 차가운 야생 속에서 치열하게 우승을 쟁취해야 합니다.

PHASE 01

30분 제한 야생 파밍

전투 시작 전, 모두에게 동일한 야생 필드가 열립니다. 주어진 단 30분이라는 시간 제한 속에 무기, 방어구, 버프 물약, 핵심 자원 및 식량을 신속하고 철저하게 확보해야 합니다.

PHASE 02

전투 시간

전투 시간이 되면 자동으로 모든 플레이어는 지상으로 tp되고 자기장은 천천이 줄어들고 모은 자원으로 PVP를 하는 시간.

PHASE 03

1대1 PVP

1대1까지 오면 거의 자기장은 0 0 쪽에 아주 가까운 상태이며 플레이어는 아주 잘 보이게 됨니다.

PHASE 04

자원 시간 팁

자원시간에는 모든 피해를 무시 , 광물를 부시면 자동으로 주괴로 나옴 , 레드스톤이 다이아로 변경됨

SERVER RULES

안전하고 정당한 플레이를 위한 필독 약관

1. 운영 및 최종 권한 (Administration)

  • 운영진 결정의 최종성: 서버 내 모든 상황에 대한 최종 판단은 운영진이 내립니다. 운영진은 서버의 안녕과 질서를 위해 필요한 경우 규칙에 명시되지 않은 사안이라도 즉각적인 조치를 취할 권한이 있습니다.
  • 규칙 회피 및 허점 악용 금지: 규칙의 문구상 허점을 찾아내어 교묘하게 위반하거나, 제재를 피하려는 모든 시도를 금지합니다. "규칙에 없었는데요?"라는 식의 변명은 통용되지 않으며, 모든 유저는 규칙의 '의도'를 파악하고 준수해야 합니다. 회피 의도가 다분할 경우 일반 위반보다 가중 처벌됩니다.

2. PVP 및 티밍 규정 (Combat)

  • 티밍(Teaming) 엄금: 개인전 기반입니다. 직접적인 동맹뿐만 아니라, 특정 대상을 고의로 공격하지 않거나 아이템을 공유하는 등의 '간접적 티밍'도 적발 시 처벌됩니다.
  • 전투 회피 금지: 전투 중 로그아웃(데스 로그), 지형지물 악용 무적 버그 활용 등을 엄격히 금지합니다.

3. 수위 및 이미지 제한 (Content)

  • 수위 가이드라인: 15세 이용가를 상회하는 선정적인 이미지(야짤), 텍스트, 링크 공유를 절대 금지합니다.
  • 우회 게시 금지: 특수문자나 외부 링크를 이용해 수위 제한을 피하려는 시도는 즉시 영구 밴 대상입니다.

4. 제재 안내 (Penalty)

  • 무관용 원칙: 핵 사용, 19금 이미지 게시, 심각한 패드립은 경고 없이 즉시 영구 밴 처리됩니다.
  • 가중 처벌: 위반 사실을 은폐하거나 운영진을 기만할 경우 처벌 수위가 대폭 상승합니다.

5. FOPVPX 서버 특수 세부 규칙

  • 액스레이 금지: 액스레이를 사용 시 서버에 영구 밴 대상이 됩니다.
  • 핵 금지: 핵을 사용 시 조치 없이 바로 영구 밴 대상이 됩니다.
  • 티밍 금지: 유저들끼리 경쟁하여 1등을 가리는 서버입니다. 지인 및 친구와의 전투 회피 또는 암묵적인 팀플레이 동맹 적발 시 [경고 2회]를 부여합니다.
  • 허위 신고 금지: 악의적이거나 거짓 정보를 활용하여 무고한 신고를 진행할 경우 [경고 1회]가 부과됩니다.
  • 채팅 언어 청결 준수: 전체 채팅창 내 욕설, 패드립, 인격 모독적 발언 및 비속어 사용이 전면 일체 차단됩니다.
